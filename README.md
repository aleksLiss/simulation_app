# Проект “Симуляция”
___ 
Реализован в рамках:

https://zhukovsd.github.io/java-backend-learning-course/projects/simulation/
___

Краткое описание:

Пошаговая симуляция 2D мира, 
населённого травоядными и хищниками. 
Кроме существ, мир содержит ресурсы (траву), 
которыми питаются травоядные, и статичные объекты, 
с которыми нельзя взаимодействовать - они просто занимают место.

2D мир представляет из себя матрицу NxM, 
каждое существо или объект занимают клетку целиком, 
нахождение в клетке нескольких объектов/существ - недопустимо.
